  • 好房網APP搜尋功能規劃

    2020/4

    wireframe密碼：housefun

  • APP搜尋功能規劃

    負責項目

    1. 產品功能企劃

      1. 依APP 評論內容以及競品分析提案，增加進階搜尋功能
      2. 調整搜尋機制與內容
      3. 提昇用戶體驗

  • 專案執行

    1. 前期：競品研究 / 蒐集數據分析 / 需求討論 / 制定wireframe /定義範圍 / UI確認 / 功能規格制定 / 確認時程
    2. 中期：定期進度檢視 / 議題討論 / 時程確認
    3. 後期：測試驗收 / 規劃階段任務 / 上線
    4. 營運：觀察評價 / APP 推播宣傳

  • 功能特點

    1. 瀏覽紀錄：儲存單機瀏覽紀錄-可藉由深度查訪物件，提供紀錄尋找清單
    2. 搜尋紀錄：儲存曾經搜尋內容-輔助記憶曾搜尋的物件關鍵字
    3. 熱搜標籤：查詢近期熱門搜尋內容，也可藉由熱搜結果帶動週邊物件能見度（單文頁的推薦物件推廣）

  • 上線時間

  • 2020/5/26 完成上線

  • 原因說明：
    規劃功能一併加入「捲動計數」功能，在技術成本與效益評估未取得平衡

    改善方向：
    團隊能認同額外新功能開發，但實質效益上並無法有確切數字提供佐證，溝通上花費時間後決議暫緩開發，應增加功能面與成本評估的說明效益，若無法有商業目標數據，減少額外的成本支出，才能有效達成迭代目標。
