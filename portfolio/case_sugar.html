  • 提昇指標效益功能規劃

    2020/7

    M版 wireframe

  • 提昇網站使用者留言數量，幫助加盟主與用戶能運用地理位置、總價、類型相符的條件，推薦物件，增加留言量，並帶入其他頻道行銷宣傳點。

    負責項目

    1. 產品功能企劃

      1. 必須符合條件相符的物件，依照POI計算範圍800公尺的距離，落於物件總價正負15％內的同類型（住宅、商辦等條件）的物件
      2. 留言後的推薦必須也要提供查看物件功能。
      3. 須排除相同物件(平台營運限制)
      4. 利用後續行為操作，宣傳推廣新頻道綁定Line會員。
      5. 依據流量觀察，將以M版優先開發，並接續Web版本功能上線。

    2. 專案執行

      1. 前期：發起需求討論 / 數據檢視 / 制定wireframe /定義範圍 / UI確認 / 確認時程
      2. 中期：定期進度檢視 / 問題討論 / 時程確認 / 拆分平台開發階段
      3. 後期：測試驗收 / 上線
      4. 營運：數據檢視 / 進行接續階段開發

    3. 功能特點-M版

      1. 留言畫面
      2. M版留言後推薦畫面
      3. M版推薦物件一併留言成功畫面
      4. 附近沒有符合可以推薦物件的畫面
      5. 只想預約一個物件的畫面

    4. 功能特點-Web版

      1. 留言畫面
      2. M版留言後推薦畫面
      3. M版推薦物件一併留言成功畫面
      4. 附近沒有符合可以推薦物件的畫面
      5. 只想預約一個物件的畫面

    5. 開發成果

      1. 依條件推薦內容，上線三週前後比較，每週增加9~12%指標數據，提高轉換率。
      2. 綁定推廣頻道數量，兩週成長5％會員數。
