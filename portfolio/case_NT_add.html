  • 新北市門牌

    2019/3

  • 瀏覽技術文章看到此連結除了新北市LOGO與下方特徵使用圖片外，用CSS畫一張門牌嘗試各種裝置對應的寫法。

  • 歷程

    1. UI KIT

    2. 製作範例

    3. 字型

      中文字體以微軟正黑體為主，大小則比對門牌比例規範

    4. 顏色

      取現有背景、文字顏色

    5. 圖片

      圖片利用Photoshop去背，當成元素使用。

    6. 考量門牌網頁中權重，雖然門牌號碼字號最大，但住址為由前至後帶入，故於KIT上會以行政區>路名>巷弄>號的方式排序。
    7. 調整字體僅能使用100%定義初始值，各字號再以標籤定義vw設定其他尺寸的大小。
    8. 門牌號碼斜體字若用italic太突兀，改採transform調整10度的樣式。
