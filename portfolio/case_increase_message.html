  • 好房網通知中心功能規劃

    2020/7

  • 市場策略調整，籌備成立新建案頻道投入新建案市場，完成各項網站企劃/籌備/建置。

    負責項目

    1. 網站頻道企劃

      1. 前/後台功能規劃
      2. 機制數據面設計
      3. 各項建商資料串接規劃
      4. 運用使用者軌跡，產出符合用戶所在地的建案內容
      5. 依建案資料規劃推薦功能

    2. 新建案內容招募

      1. 營運前/後-招募建案資料流程規劃
      2. 提供用戶重點資訊內容
      3. <創新>加入房價資訊對比內容
      4. 提供建商刊登聯繫，協助建案量成長

    3. 功能規劃

      1. 利用區域推薦附近新建案內容
      2. 標註特色tag，快速了解建案特徵
      3. 首創建案比較功能，標示/隱藏標的建案
      4. 串接電話系統，個人資訊不外流

    4. 維運

      1. 依據資料端需求，逐步調整資訊呈現方式
      2. 提供合作廠商端各項指標數據報表
      3. 建案地址與接待中心地圖資料校正
      4. 逐步調整SEO內容
