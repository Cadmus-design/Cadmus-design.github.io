進階型產品介紹

  • 順暢單頻版 802.11n

    • 外接式全向型天線：2支5dBi高增益天線，提供更穩定無線訊號
    • 網路安全：具備DoS防護安全的功能，居家網路更安全

  • 覆蓋雙頻版(標準) Mesh 802.11ac

    • 高效能傳輸：高效能處理網路流量，擴展無線網路覆蓋範圍
    • 直立式外觀設：搭配隱藏式天線，節省空間、容易擺放
    • 建議環境：單樓層住宅/公寓(無跨樓層之場域)
    • 支援同步雙頻(2.4GHz+5GHz)，傳輸效率更高

  • 覆蓋雙頻版 Mesh 802.11ac

    • 具備「Mesh」網狀架構功能的Wi-Fi AP，至少要2台(含)以上同廠牌型號Mesh Wi-Fi AP始能組成Mesh Wi-Fi網路架構
    • 免破壞裝潢：多台Mesh Wi-Fi AP間可自動連線無須拉線（部分場域仍須拉實體線，以現場評估為準）即可完成家中Wi-Fi網路環境建置（惟主機AP須與數據機/外接式 AP(具PPPoE功能)以實體網路線連接，副機AP接電即可）
    • 更好的使用者體驗：單一無線網路名稱(SSID)，不需手動切換SSID。且具漫遊機制，可使用戶移動過程中的「跨AP漫遊」更加順暢
    • 訊號最佳化：自動將聯網裝置連線至擁有最強Wi-Fi訊號的AP
    • 提升Wi-Fi訊號涵蓋率：用戶可依家中環境需求加裝AP來擴大Wi-Fi訊號覆蓋範圍
    • 透天厝/樓中樓/大坪數住家環境建置穩定Wi-Fi的最佳方案
